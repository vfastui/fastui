# 提示

## 基本用法
<FTooltip tooltip="这里是提示内容"><FButton>默认</FButton></FTooltip>
```html
<FTooltip tooltip="这里是提示内容">
    <FButton>默认</FButton>
</FTooltip>    
```

## 文字提示  

<FTooltip tooltip="这里是提示内容" placement="bottom-center" theme="light">文字</FTooltip>

## 主题
`theme` ： `dark`,`light`

## 位置
`placement` ：`top-left`,
`top-center`,
`top-right`,
`bottom-left`,
`bottom-center`,
`bottom-right`,
`left-top`,
`left-center`,
`left-bottom`,
`right-top`,
`right-center`,
`right-bottom`
